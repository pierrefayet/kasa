
# KASA

Kasa is an apartment rental site. 

This project has been implemented with React 18.2.0.
## Run Locally

Clone the project

```bash
  git clone git@github.com:pierrefayet/kasa.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```
Start the server

```bash
  npm start
```
